## 📌 Project Overview

This project builds a full ML pipeline that predicts whether a patient is at risk of developing a **chronic disease** based on 13 clinical and lifestyle features. It covers everything from raw data ingestion to a deployed Streamlit web app.

**Target Column**: `HasChronicDisease` (Yes / No → 1 / 0)

---

## 📊 Dataset

| Property | Value |
|---|---|
| Rows | 1,500 |
| Features | 13 (after dropping Patient_ID) |
| Target | HasChronicDisease |
| Class Balance | 58.6% No  /  41.4% Yes |
| Missing Values | None |

**Features include**: Age, Gender, BMI, Smoking, Alcohol Intake, Physical Activity, Diet Quality, Sleep Hours, Blood Pressure, Cholesterol, Glucose, Family History, Stress Level

## 🚀 Quick Start

```bash
# 1. Clone the repository
git clone https://github.com/YOUR_USERNAME/chronic-disease-prediction.git
cd chronic-disease-prediction

# 2. Create & activate a virtual environment
python -m venv venv
source venv/bin/activate        # Windows: venv\Scripts\activate

# 3. Install dependencies
pip install -r requirements.txt

# 4. Run the full ML pipeline (generates models + plots)
python chronic_disease_model.py

# 5. Launch the Streamlit web app
streamlit run app.py
```

## 🔬 Pipeline Steps

| Step | Description |
|------|-------------|
| **1. Imports** | numpy, pandas, scikit-learn, matplotlib, seaborn, joblib |
| **2. Data Cleaning** | Null handling, duplicate removal, type corrections |
| **3. Visualisation** | Distribution plots, Correlation Heatmap |
| **4. EDA** | Box plots (numeric vs target), Stacked bar (categorical vs target) |
| **5. Model Choice** | Gradient Boosting Classifier (GBM) |
| **6. Feature/Target Split** | X (21 features after OHE) / y |
| **7. Train-Test Split** | 80-20, stratified |
| **8. Model Training** | GBM with tuned hyperparameters |
| **9. Evaluation** | Accuracy, Classification Report, Confusion Matrix |
| **10. ROC Curve** | AUC = 1.0000 |
| **11. Model Saving** | `.sav` (Joblib format) |
| **12. Pipeline** | ColumnTransformer + Scaler + Encoder + GBM |
| **13. Pipeline Saving** | Compressed `.pkl.gz` |
